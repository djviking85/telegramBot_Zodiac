# Телеграмм Бот-горокоп
Коммерческий проект для телеграмм бота на основе Java

## Зависимости и тех. часть:
- SpringFramework
- - starter
- -  contex support
- postgresql
- telegrambots
- Java ver. 11
- projectlombok
- maven
## Функционал бота:
- кнопочное распределение бота
- введение различных дат, для определения знаков
